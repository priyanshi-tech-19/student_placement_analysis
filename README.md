# Student Placement Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a student placement dataset using Python and Pandas. The objective is to analyze placement trends, salary distribution, academic performance, internships, GitHub activity, and other factors affecting student placements.

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Analysis Performed
- Data Inspection
- Filtering & Sorting
- GroupBy & Aggregation
- Salary Analysis
- Placement Analysis
- Correlation Analysis
- Key Insights
